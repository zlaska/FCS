# 灵活就业共创规范 FCS
**Flexible Cooperation Standard**

> 一份让闲置时间不被浪费、让有想法的人不至孤军奋战的协作默认规则。

[![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-sa/4.0/)

---

## 这是什么

FCS 是一份 **协作的默认规则**——当一群人临时聚到一起做点事时（项目还没公司化），可以共同声明"我们采用 FCS"，省去每次重新讨论"贡献怎么算、利益怎么分、人怎么进出"。

它面向：失业者、自由职业者、有空闲者、想要转行或投身公益事业的人。

## 它不是什么

- 不是公司章程
- 不是法律合约
- 不是替代律师的法律文件

它是一份 **轻量协作框架**，覆盖项目从"几个人聚一起做事"到"决定公司化"的中间地带。

## 核心特点

- ⏱ **1 小时 = 1 点** —— 不分职位高低，劳动时间等价
- 🔄 **动态分配** —— 谁多做谁多得
- 🚪 **自由进出** —— 退出不清零，加入不需要"买入"
- 🗳 **简单治理** —— 大多数事简单多数决，少数关键事 4/5
- 📜 **可自定义** —— 团队可在内部投票调整任何条款

## 快速开始

1. 团队达成共识："我们采用 FCS v1.0"
2. 由发起人写一份 **合作愿景**（要做什么、为什么做）
3. 找一个支持 FCS 规则的平台（或自建表格），开始记录贡献
4. 完整规则见 [`灵活就业共创规范.md`](./灵活就业共创规范.md)

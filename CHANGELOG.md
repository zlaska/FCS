# CHANGELOG

本文件记录 FCS 协议的版本变更。版本号遵循语义化版本规则（参见 [CONTRIBUTING.md](./CONTRIBUTING.md)）。

---

## v1.0 — 2026-06-12

首个正式版本。

### 包含的核心章节
- 宗旨
- 合作愿景
- 团队组建与流动
- 贡献度量（劳动时间 / 资金 / 生产材料）
- 任务认领与分配
- 验收机制（含质疑机制）
- 决策机制（推荐流程）
- 争议与欺诈处理
- 投票与兜底
- 知识产权与想法
- 边界（处理范围 / 结算与升级 / 后续发展建议）

### 关键设计决定
- 1 小时 = 1 点，职位不分高低
- 资金权重建议为 3，可团队投票调整
- 贡献点永久归属，退出不清零
- 追加投资需现有成员简单多数同意
- 修改合作愿景与认定欺诈需 4/5 多数
- 知识产权随个人退出而失效
